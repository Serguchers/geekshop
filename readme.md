# Интернет-магазин на основе Django

Учебный проект интернет-магазина, который разработан при прохождений обучения на факультете "Python-разработка", портал GeekBrains, на курсе `Django Framework`.

## Стек

* `Python` > 3.5
* `Django` < 3.0
* `VSCode`
* `SQLite3`
